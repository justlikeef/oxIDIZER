# ox_cert_core — Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Prerequisite:** Plan 00 (ox_persistence enhancements) must be complete before this plan.

**Goal:** Build the `ox_cert_core` shared library — all common types, CertError, software KeyStore, CertStore backed by ox_persistence, X.509 and SSH certificate builders, issuance policy engine, OCSP utilities, CT submission, and messaging helpers. Every cert plugin depends on this crate.

**Architecture:** A single `lib` crate (not cdylib). Ships as a path dependency to all cert plugin crates. Wraps `rcgen` for X.509 generation, `ssh-key` for OpenSSH certificates, `ring` for low-level crypto (HKDF, AES-GCM), and `ox_persistence` for storage via GDO/ORM. The CertStore trait isolates all persistence behind a testable interface; `OxPersistenceCertStore` is the concrete implementation.

**Tech Stack:** Rust 2021, rcgen 0.13, ring 0.17, x509-parser 0.16, ssh-key 0.6, p12 0.6, hkdf 0.12, sha2 0.10, base64 0.22, uuid 1.6 (v4), serde 1.0, time 0.3, thiserror 1.0, regex 1.10, reqwest 0.12 (blocking + json), ox_persistence (path dep), ox_workflow_abi (path dep), libc 0.2.

```rust
use hkdf::Hkdf;
use sha2::Sha256;
use ring::aead::{self, BoundKey, Aad, UnboundKey, AES_256_GCM, NONCE_LEN};
use ring::rand::{SecureRandom, SystemRandom};
use base64::{Engine, engine::general_purpose::STANDARD as B64};
use crate::error::CertError;

const INFO: &[u8] = b"ox_cert:private_key_enc_v1";

/// Derive AES-256-GCM key from OX_CA_KEY_PASS env var + tenant_id salt.
pub fn derive_key(tenant_id: &str) -> Result<[u8; 32], CertError> {
    let ikm = std::env::var("OX_CA_KEY_PASS")
        .map_err(|_| CertError::ConfigError("OX_CA_KEY_PASS not set".into()))?;
    let hk = Hkdf::<Sha256>::new(Some(tenant_id.as_bytes()), ikm.as_bytes());
    let mut okm = [0u8; 32];
    hk.expand(INFO, &mut okm)
        .map_err(|e| CertError::Internal(format!("HKDF expand: {e}")))?;
    Ok(okm)
}

/// Encrypt plaintext → base64(nonce[12] || ciphertext || tag[16]).
pub fn encrypt_private_key(plaintext: &[u8], tenant_id: &str) -> Result<String, CertError> {
    let key_bytes = derive_key(tenant_id)?;
    let rng = SystemRandom::new();
    let mut nonce_bytes = [0u8; NONCE_LEN];
    rng.fill(&mut nonce_bytes)
        .map_err(|_| CertError::Internal("RNG failed".into()))?;

    let unbound = UnboundKey::new(&AES_256_GCM, &key_bytes)
        .map_err(|_| CertError::Internal("AES key init failed".into()))?;
    let nonce = aead::Nonce::assume_unique_for_key(nonce_bytes);
    let mut sealing_key = aead::SealingKey::new(unbound, aead::OneNonceSequence::new(nonce));

    let mut in_out = plaintext.to_vec();
    sealing_key.seal_in_place_append_tag(Aad::empty(), &mut in_out)
        .map_err(|_| CertError::Internal("AES-GCM seal failed".into()))?;

    let mut result = nonce_bytes.to_vec();
    result.extend_from_slice(&in_out);
    Ok(B64.encode(&result))
}

/// Decrypt base64(nonce || ciphertext || tag) → plaintext bytes.
pub fn decrypt_private_key(encoded: &str, tenant_id: &str) -> Result<Vec<u8>, CertError> {
    let key_bytes = derive_key(tenant_id)?;
    let raw = B64.decode(encoded)
        .map_err(|e| CertError::Internal(format!("base64 decode: {e}")))?;
    if raw.len() < NONCE_LEN {
        return Err(CertError::Internal("ciphertext too short".into()));
    }
    let (nonce_bytes, ciphertext) = raw.split_at(NONCE_LEN);
    let mut nonce = [0u8; NONCE_LEN];
    nonce.copy_from_slice(nonce_bytes);

    let unbound = UnboundKey::new(&AES_256_GCM, &key_bytes)
        .map_err(|_| CertError::Internal("AES key init failed".into()))?;
    let nonce = aead::Nonce::assume_unique_for_key(nonce);
    let mut opening_key = aead::OpeningKey::new(unbound, aead::OneNonceSequence::new(nonce));

    let mut in_out = ciphertext.to_vec();
    let plaintext = opening_key.open_in_place(Aad::empty(), &mut in_out)
        .map_err(|_| CertError::Internal("AES-GCM open failed — wrong key or corrupt data".into()))?;
    Ok(plaintext.to_vec())
}
```

> **Note on ring API**: `ring::aead` uses `OneNonceSequence` with `SealingKey`/`OpeningKey`. If the ring version in use has a different API (e.g., `LessSafeKey`), adapt accordingly — the semantics are identical.

- [ ] **Step 3: Implement `SoftwareKeyStore`**

Key path pattern: `{key_dir}/{tenant_id}/{key_id}.key.pem`

```rust
pub struct SoftwareKeyStore { config: KeyStoreConfig }

impl SoftwareKeyStore {
    fn key_path(&self, tenant_id: &str, key_id: &str) -> PathBuf {
        self.config.key_dir.as_ref().unwrap()
            .join(tenant_id)
            .join(format!("{key_id}.key.pem"))
    }
}

impl KeyStore for SoftwareKeyStore {
    fn open(config: &KeyStoreConfig) -> Result<Self, CertError> {
        if let Some(dir) = &config.key_dir {
            std::fs::create_dir_all(dir)
                .map_err(|e| CertError::KeyStoreError(format!("create key_dir: {e}")))?;
        }
        Ok(Self { config: config.clone() })
    }

    fn generate_key(&self, tenant_id: &str, key_id: &str, key_type: KeyType, overwrite: bool)
        -> Result<(), CertError>
    {
        let path = self.key_path(tenant_id, key_id);
        if path.exists() && !overwrite { return Ok(()); }
        std::fs::create_dir_all(path.parent().unwrap())
            .map_err(|e| CertError::KeyStoreError(e.to_string()))?;

        // Generate PKCS#8 PEM using rcgen's KeyPair
        let kp = match key_type {
            KeyType::EcP256 => rcgen::KeyPair::generate_for(&rcgen::PKCS_ECDSA_P256_SHA256),
            KeyType::EcP384 => rcgen::KeyPair::generate_for(&rcgen::PKCS_ECDSA_P384_SHA384),
            KeyType::Ed25519 => rcgen::KeyPair::generate_for(&rcgen::PKCS_ED25519),
            KeyType::Rsa2048 | KeyType::Rsa3072 | KeyType::Rsa4096 => {
                // ring does not expose RSA key generation; use rcgen's built-in
                rcgen::KeyPair::generate_for(&rcgen::PKCS_RSA_SHA256)
            }
            _ => return Err(CertError::KeyStoreError(format!("unsupported key type: {key_type:?}"))),
        }.map_err(|e| CertError::KeyStoreError(format!("generate: {e}")))?;

        std::fs::write(&path, kp.serialize_pem())
            .map_err(|e| CertError::KeyStoreError(format!("write key: {e}")))?;
        Ok(())
    }

    fn sign(&self, tenant_id: &str, key_id: &str, _algo: SigningAlgorithm, data: &[u8])
        -> Result<Vec<u8>, CertError>
    {
        let path = self.key_path(tenant_id, key_id);
        let pem = std::fs::read_to_string(&path)
            .map_err(|_| CertError::KeyStoreError(format!("key not found: {key_id}")))?;
        let kp = rcgen::KeyPair::from_pem(&pem)
            .map_err(|e| CertError::KeyStoreError(format!("parse key: {e}")))?;
        // rcgen KeyPair can sign raw data via ring
        kp.sign(data).map_err(|e| CertError::KeyStoreError(format!("sign: {e}")))
    }

    fn public_key(&self, tenant_id: &str, key_id: &str) -> Result<Vec<u8>, CertError> {
        let path = self.key_path(tenant_id, key_id);
        let pem = std::fs::read_to_string(&path)
            .map_err(|_| CertError::KeyStoreError(format!("key not found: {key_id}")))?;
        let kp = rcgen::KeyPair::from_pem(&pem)
            .map_err(|e| CertError::KeyStoreError(format!("parse key: {e}")))?;
        Ok(kp.public_key_der())
    }

    fn key_exists(&self, tenant_id: &str, key_id: &str) -> Result<bool, CertError> {
        Ok(self.key_path(tenant_id, key_id).exists())
    }

    fn delete_key(&self, tenant_id: &str, key_id: &str) -> Result<(), CertError> {
        let path = self.key_path(tenant_id, key_id);
        if path.exists() {
            std::fs::remove_file(&path)
                .map_err(|e| CertError::KeyStoreError(format!("delete: {e}")))?;
        }
        Ok(())
    }

    fn key_info(&self, tenant_id: &str, key_id: &str) -> Result<KeyInfo, CertError> {
        let path = self.key_path(tenant_id, key_id);
        let meta = std::fs::metadata(&path)
            .map_err(|_| CertError::KeyStoreError(format!("key not found: {key_id}")))?;
        Ok(KeyInfo {
            key_id: key_id.to_string(),
            tenant_id: tenant_id.to_string(),
            key_type: KeyType::EcP256, // TODO: parse from PEM
            created_at: time::OffsetDateTime::now_utc(),
        })
    }
}
```

```rust
use crate::certstore::CertStoreConfig;
use crate::error::CertError;
use ox_persistence::{PERSISTENCE_DRIVER_REGISTRY, PersistenceDriver};
use serde_json::json;
use std::sync::Arc;

const MIGRATIONS: &[(&str, &str)] = &[
    ("001_initial", include_str!("001_initial.sql")),
];

/// Run all unapplied migrations. Safe to call on every startup.
pub fn run_migrations(
    driver: &Arc<dyn ox_persistence::PersistenceDriver + Send + Sync>,
) -> Result<(), CertError> {
    // Bootstrap: ensure schema_migrations table exists
    driver.call_action("run_ddl", &json!({
        "sql": "CREATE TABLE IF NOT EXISTS schema_migrations (version TEXT PRIMARY KEY, applied_at TEXT NOT NULL)"
    })).map_err(|e| CertError::StorageError(format!("bootstrap migrations table: {e}")))?;

    for (version, sql) in MIGRATIONS {
        // Check if already applied
        let rows = driver.call_action("raw_sql", &json!({
            "sql": "SELECT version FROM schema_migrations WHERE version = $1",
            "params": [version]
        })).map_err(|e| CertError::StorageError(e.to_string()))?;

        if rows["rows"].as_array().map(|a| !a.is_empty()).unwrap_or(false) {
            continue; // already applied
        }

        // Run the migration SQL (may contain multiple statements)
        driver.call_action("run_ddl", &json!({ "sql": sql }))
            .map_err(|e| CertError::StorageError(format!("migration {version}: {e}")))?;

        // Record as applied
        let now = time::OffsetDateTime::now_utc().to_string();
        driver.call_action("run_ddl", &json!({
            "sql": format!("INSERT INTO schema_migrations (version, applied_at) VALUES ('{version}', '{now}')")
        })).map_err(|e| CertError::StorageError(format!("record migration {version}: {e}")))?;
    }
    Ok(())
}
```

```rust
use regex::Regex;
use crate::types::{IssuancePolicyConfig, CsrInfo, SanType, KeyType};
use crate::error::CertError;

pub struct IssuancePolicy {
    pub domain_allowlist: Vec<Regex>,
    pub domain_blocklist: Vec<Regex>,
    pub max_san_count: usize,
    pub wildcard_allowed: bool,
    pub min_rsa_bits: u32,
    pub require_ra_approval: bool,
}

impl IssuancePolicy {
    pub fn from_config(cfg: &IssuancePolicyConfig) -> Result<Self, CertError> {
        let allowlist = cfg.domain_allowlist.iter()
            .map(|s| Regex::new(s).map_err(|e| CertError::ConfigError(format!("bad allowlist regex: {e}"))))
            .collect::<Result<Vec<_>, _>>()?;
        let blocklist = cfg.domain_blocklist.iter()
            .map(|s| Regex::new(s).map_err(|e| CertError::ConfigError(format!("bad blocklist regex: {e}"))))
            .collect::<Result<Vec<_>, _>>()?;
        Ok(Self {
            domain_allowlist: allowlist,
            domain_blocklist: blocklist,
            max_san_count: cfg.max_san_count,
            wildcard_allowed: cfg.wildcard_allowed,
            min_rsa_bits: cfg.min_rsa_bits,
            require_ra_approval: cfg.require_ra_approval,
        })
    }

    pub fn validate_csr(&self, csr: &CsrInfo) -> Result<(), CertError> {
        if csr.sans.len() > self.max_san_count {
            return Err(CertError::PolicyViolation(
                format!("SAN count {} exceeds maximum {}", csr.sans.len(), self.max_san_count)));
        }
        if matches!(csr.key_type, KeyType::Rsa2048 | KeyType::Rsa3072 | KeyType::Rsa4096) {
            if csr.key_bits < self.min_rsa_bits {
                return Err(CertError::PolicyViolation(
                    format!("RSA key {} bits < minimum {}", csr.key_bits, self.min_rsa_bits)));
            }
        }
        for san in &csr.sans {
            if let SanType::Dns(name) = san {
                if name.starts_with("*.") && !self.wildcard_allowed {
                    return Err(CertError::PolicyViolation(format!("wildcard SANs not allowed: {name}")));
                }
                // Must match at least one allowlist pattern (if allowlist is non-empty)
                if !self.domain_allowlist.is_empty() {
                    let allowed = self.domain_allowlist.iter().any(|r| r.is_match(name));
                    if !allowed {
                        return Err(CertError::PolicyViolation(format!("domain not in allowlist: {name}")));
                    }
                }
                // Must not match any blocklist pattern
                for block in &self.domain_blocklist {
                    if block.is_match(name) {
                        return Err(CertError::PolicyViolation(format!("domain is blocked: {name}")));
                    }
                }
            }
        }
        Ok(())
    }
}
```

```rust
pub struct ChainBuilder { root_cert_pem: String, intermediate_cert_pem: String }

impl ChainBuilder {
    pub fn from_pem(root: &str, intermediate: &str) -> Result<Self, CertError> {
        Ok(Self { root_cert_pem: root.to_string(), intermediate_cert_pem: intermediate.to_string() })
    }
    pub fn from_paths(root: &std::path::Path, intermediate: &std::path::Path) -> Result<Self, CertError> {
        let root_pem = std::fs::read_to_string(root).map_err(|e| CertError::ConfigError(e.to_string()))?;
        let int_pem = std::fs::read_to_string(intermediate).map_err(|e| CertError::ConfigError(e.to_string()))?;
        Self::from_pem(&root_pem, &int_pem)
    }
    pub fn full_chain_pem(&self, issued: &str) -> String {
        format!("{}\n{}\n{}", issued.trim(), self.intermediate_cert_pem.trim(), self.root_cert_pem.trim())
    }
    pub fn ca_chain_pem(&self) -> String {
        format!("{}\n{}", self.intermediate_cert_pem.trim(), self.root_cert_pem.trim())
    }
}
```

```rust
// ocsp.rs
pub fn ocsp_serial_to_uuid(der_bytes: &[u8]) -> Result<String, CertError> {
    if der_bytes.len() != 16 {
        return Err(CertError::InvalidCsr(format!("OCSP serial must be 16 bytes, got {}", der_bytes.len())));
    }
    let uuid = uuid::Uuid::from_bytes(der_bytes.try_into().unwrap());
    Ok(uuid.to_string())
}

pub fn uuid_to_ocsp_serial(uuid_str: &str) -> Result<Vec<u8>, CertError> {
    let uuid = uuid::Uuid::parse_str(uuid_str)
        .map_err(|e| CertError::InvalidCsr(format!("invalid serial UUID: {e}")))?;
    Ok(uuid.as_bytes().to_vec())
}
```
