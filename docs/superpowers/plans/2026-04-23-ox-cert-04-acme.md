# ox_cert ACME Protocol — Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Prerequisite:** Plans 00, 01, and 02 must be complete.

**Goal:** Build three cdylib plugins implementing RFC 8555 ACME: `ox_cert_acme` (main ACME server — accounts, orders, authorizations, finalization), `ox_cert_acme_challenge_http` (HTTP-01 challenge validator), and `ox_cert_acme_challenge_dns` (DNS-01 challenge validator).

**Architecture:** `ox_cert_acme` handles all ACME routes and sets challenge context in TaskState. The challenge plugins run downstream in the same Content phase and read that context. All three are independent cdylibs. Nonce management is either in-memory (single node) or database-backed (multi-node). JWS verification uses `josekit`.

**Tech Stack:** Rust 2021 cdylib, ox_cert_core, josekit 0.8, uuid 1.6, base64 0.22, sha2 0.10, serde_json, time 0.3, reqwest 0.12 (blocking, for HTTP-01 and DNS provider API), trust-dns-resolver 0.23 (DNS-01).

```rust
impl MemoryNonceStore {
    pub fn new(ttl: std::time::Duration) -> Self {
        Self { nonces: std::sync::RwLock::new(std::collections::HashMap::new()), ttl }
    }

    pub fn issue(&self) -> String {
        let nonce = uuid::Uuid::new_v4().to_string();
        self.nonces.write().unwrap().insert(nonce.clone(), std::time::Instant::now());
        nonce
    }

    pub fn consume(&self, nonce: &str) -> bool {
        let mut map = self.nonces.write().unwrap();
        if let Some(issued_at) = map.remove(nonce) {
            issued_at.elapsed() < self.ttl
        } else {
            false
        }
    }

    pub fn cleanup_expired(&self) {
        let mut map = self.nonces.write().unwrap();
        let ttl = self.ttl;
        map.retain(|_, issued_at| issued_at.elapsed() < ttl);
    }
}
```

```rust
// jws.rs
use josekit::jws::{JwsHeader, deserialize_compact};

pub struct VerifiedJws {
    pub payload: Vec<u8>,
    pub account_id: Option<String>,  // from kid
    pub jwk: Option<josekit::jwk::Jwk>,  // from embedded jwk
    pub nonce: String,
}

/// Verify a JWS body. Returns the payload and identity info.
pub fn verify_jws(
    raw_body: &str,
    expected_url: &str,
    store: &dyn CertStore,
    tenant_id: &str,
    nonce_store: &NonceStore,
) -> Result<VerifiedJws, CertError> {
    // 1. Parse compact JWS: split header.payload.signature
    // 2. Decode protected header
    let (payload, header) = deserialize_compact_insecure(raw_body)
        .map_err(|e| CertError::AcmeError(format!("malformed JWS: {e}")))?;
    // 3. Verify nonce
    let nonce = header.nonce().ok_or_else(|| CertError::AcmeError("missing nonce".into()))?;
    if !nonce_store.consume(nonce) {
        return Err(CertError::AcmeError("badNonce".into()));
    }
    // 4. Verify URL
    if header.url().unwrap_or("") != expected_url {
        return Err(CertError::AcmeError("URL mismatch".into()));
    }
    // 5. Verify signature
    if let Some(kid) = header.key_id() {
        // kid = account URL → load account → use account.jwk
        let account_id = kid.split('/').last().unwrap_or("").to_string();
        let account = store.get_acme_account(tenant_id, &account_id)?
            .ok_or_else(|| CertError::NotFound(format!("account {account_id}")))?;
        let jwk: josekit::jwk::Jwk = serde_json::from_str(&account.jwk)
            .map_err(|e| CertError::AcmeError(format!("invalid account JWK: {e}")))?;
        verify_compact_with_jwk(raw_body, &jwk)
            .map_err(|e| CertError::AcmeError(format!("signature invalid: {e}")))?;
        Ok(VerifiedJws { payload, account_id: Some(account_id), jwk: None, nonce: nonce.to_string() })
    } else if let Some(jwk) = header.jwk() {
        // Embedded JWK (new-account only)
        verify_compact_with_jwk(raw_body, jwk)
            .map_err(|e| CertError::AcmeError(format!("signature invalid: {e}")))?;
        Ok(VerifiedJws { payload, account_id: None, jwk: Some(jwk.clone()), nonce: nonce.to_string() })
    } else {
        Err(CertError::AcmeError("JWS must have kid or jwk".into()))
    }
}
```

```rust
pub fn handle_cert_download(ctx: &ModuleContext, order_id: &str, task: &TaskState) -> Result<(), CertError> {
    let order = ctx.store.get_acme_order(&ctx.config.tenant_id, order_id)?
        .ok_or_else(|| CertError::NotFound(order_id.to_string()))?;
    if order.status != AcmeOrderStatus::Valid {
        return Err(CertError::AcmeError(format!("order status is {:?}, not valid", order.status)));
    }
    let serial = order.certificate_serial.ok_or_else(|| CertError::Internal("missing cert serial".into()))?;
    let cert = ctx.store.get_cert_by_serial(&ctx.config.tenant_id, &serial)?
        .ok_or_else(|| CertError::NotFound(serial.clone()))?;
    let chain = ChainBuilder::from_paths(
        std::path::Path::new(&ctx.config.ca_root_cert_path),
        std::path::Path::new(&ctx.config.ca_intermediate_cert_path),
    )?.full_chain_pem(&cert.pem);
    task.set("response.status", "200");
    task.set("response.body", &chain);
    task.set("response.header.Content-Type", "application/pem-certificate-chain");
    Ok(())
}
```

```rust
fn process(ctx: &ModuleContext, task: &TaskState) -> Result<FlowControl, CertError> {

        .ok_or_else(|| CertError::NotFound(authz_id.clone()))?;

    let expected = format!("{token}.{thumbprint}");

    let valid = (0..ctx.config.retries + 1).any(|_| {
        reqwest::blocking::Client::new()
            .get(&url)
            .timeout(std::time::Duration::from_secs(ctx.config.timeout_secs))
            .send()
            .ok()
            .and_then(|r| r.text().ok())
            .map(|body| body.trim() == expected)
            .unwrap_or(false)
    });

    let new_status = if valid { AcmeAuthzStatus::Valid } else { AcmeAuthzStatus::Invalid };
    let mut updated_authz = authz.clone();
    updated_authz.status = new_status;
    ctx.store.update_acme_authorization(&ctx.config.tenant_id, &updated_authz)?;
    Ok(FLOW_CONTROL_CONTINUE)
}
```

```rust
fn process(ctx: &ModuleContext, task: &TaskState) -> Result<FlowControl, CertError> {
    if task.get("cert.acme.challenge.type") != "dns-01" {
        return Ok(FLOW_CONTROL_CONTINUE);
    }
    let token = task.get("cert.acme.challenge.token");
    let thumbprint = task.get("cert.acme.challenge.thumbprint");
    let authz_id = task.get("cert.acme.challenge.id");
    let authz = ctx.store.get_acme_authorization(&ctx.config.tenant_id, &authz_id)?

    // Expected TXT value
    use sha2::{Sha256, Digest};
    use base64::Engine;
    let key_auth = format!("{token}.{thumbprint}");
    let digest = Sha256::digest(key_auth.as_bytes());
    let expected = base64::engine::general_purpose::URL_SAFE_NO_PAD.encode(digest);

    let fqdn = format!("_acme-challenge.{}", authz.identifier_value);

    // Optional propagation delay
    if ctx.config.propagation_delay_secs > 0 {
        std::thread::sleep(std::time::Duration::from_secs(ctx.config.propagation_delay_secs));
    }

    // DNS TXT lookup using trust-dns-resolver
    let resolver = trust_dns_resolver::Resolver::new(
        trust_dns_resolver::config::ResolverConfig::default(),
        trust_dns_resolver::config::ResolverOpts::default(),
    ).map_err(|e| CertError::Internal(format!("DNS resolver: {e}")))?;

    let valid = resolver.txt_lookup(&fqdn).ok()
        .map(|records| records.iter().any(|txt| {
            txt.iter().any(|bytes| String::from_utf8_lossy(bytes) == expected)
        }))
        .unwrap_or(false);

    let new_status = if valid { AcmeAuthzStatus::Valid } else { AcmeAuthzStatus::Invalid };
    let mut updated = authz.clone();
    updated.status = new_status;
    ctx.store.update_acme_authorization(&ctx.config.tenant_id, &updated)?;
    Ok(FLOW_CONTROL_CONTINUE)
}
```
