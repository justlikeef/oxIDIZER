How to present a client certificate

To see client certificate trust information, connect using a certificate issued by this CA.

  • Install a client certificate in your browser and configure the server to require mutual TLS (mTLS).
  • Use curl --cert client.crt --key client.key https://<ca-server>/ca/trust
  • Pass the certificate PEM as the X-Client-Cert header for proxy or testing scenarios.
